# AI利用で学習 - React編
恐らく開発の核となる為、改めて理解している事や疑問をここに記載する。

## Reactとは
UI開発に特化したJavaScriptライブラリの一種。
特徴としては、生のJS開発と違い、コードをコンポーネント化して開発が出来る。

## Reactのメリット
1. 作業効率が上がる
- パーツごとに管理する事で、コードの視認性が向上する。

2. 保守性が高い
- コンポーネントは再利用できるので修正コストを削減できる。

## JavaScriptライブラリ
- react
- react-router-dom（React専用のプラグイン：遷移（配下）ページが作成出来る）
